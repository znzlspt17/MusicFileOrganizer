# Music File Organizer

이 애플리케이션은 메타데이터를 기반으로 음악 파일을 정리하는 프로그램입니다.

## 시스템 요구 사항
- Windows 10 이상
- .NET Core 3.1 이상
- 최소 2GB의 RAM

## Windows App Runtime 설치 지침
1. [Microsoft 공식 웹사이트](https://dotnet.microsoft.com/download/dotnet-core)에서 .NET Core 런타임을 다운로드합니다.
2. 다운로드한 설치 프로그램을 실행하고 설치 과정을 따릅니다.

## 빌드 지침
1. 소스 코드를 다운로드하거나 클론합니다.
2. 명령 프롬프트를 열고 프로젝트 디렉토리로 이동합니다.
3. 다음 명령어를 입력하여 애플리케이션을 빌드합니다:
   ```bash
   dotnet build
   ```

## 사용 가이드
1. 애플리케이션을 실행합니다.
2. 음악 파일이 저장된 폴더를 선택합니다.
3. 음악 파일이 정리될 폴더를 선택합니다.
4. "시작" 버튼을 클릭하여 프로세스를 시작합니다.
